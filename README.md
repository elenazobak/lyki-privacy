# lyki-privacy

Privacy policy for **LYKI** — the iOS vibe reel maker.

The policy is a single Jekyll page (`index.md`) published via GitHub Pages at
<https://elenazobak.github.io/lyki-privacy/>. That URL is the one linked from
the App Store listing and from inside the app.

## What's here

| File | Purpose |
|------|---------|
| `index.md` | The privacy policy itself (Markdown + front matter). |
| `_config.yml` | Jekyll site config (title, description, theme). |

## Editing the policy

Edit `index.md`. When the substance changes, update the `_Last updated:` date
near the top so the "Changes" section stays accurate.

## Preview locally

```sh
bundle install
bundle exec jekyll serve
```

Then open <http://localhost:4000/lyki-privacy/>. Requires Ruby and Bundler.
